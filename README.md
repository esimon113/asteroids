# Asteroids

A basic Asteroids game implementation using pygame. Built as a learning project to try out `uv` (as an alternative to pip/venv) and neovim.

## Requirements

- Python 3.11+
- pygame 2.6.1+


## Running

```bash
uv run main.py
```

## Controls

- **A/D** - Rotate left/right
- **W/S** - Move forward/backward
- **SPACE** - Shoot

## Rules 
Avoid and shoot the asteroids!
